# Dev OS

This repository is my personal engineering operating system.

Its purpose is not to store information, but to improve how I think, build and ship software.

Every document, prompt, checklist and workflow included here has been tested in real projects and refined through daily use.

## Principles

- Simplicity over complexity.
- Reuse over repetition.
- Automation over manual work.
- Understanding over memorization.
- AI as a collaborator, not a replacement.
- Continuous improvement through small iterations.

## Goal

Become a better engineer by building better systems.
